# Taller 2 - Videojuegos

Un ejercicio de desarrollar un entorno 3D, una casa y el movimiento del jugador en la casa con otras implementaciones.

## Requirements

- **Godot Version**: 4.5.1-stable
  Descargar desde: [https://godotengine.org/download/archive/4.5.1-stable/](https://godotengine.org/download/archive/4.5.1-stable/)
- **Scripting Language**: GDScript (standard)

## Project Structure

- **addons/**: Recursos de terceros, incluyendo modelos 3D de personajes y texturas, así como entornos HDR para iluminación.
- **assets/**: Recursos propios del proyecto, como efectos de sonido para pasos en diferentes superficies.
- **characters/**: Escenas y scripts de personajes jugables, incluyendo el jugador con su lógica de movimiento y máquina de estados.
- **docs/**: Documentación y guías, como las mejores prácticas de Godot.
- **game_world/**: Escenas de niveles del juego, como el nivel principal con plataformas y entorno.
- **main/**: Escena principal y script de entrada del juego.
- **scripts/**: Scripts reutilizables, como la máquina de estados del jugador.
